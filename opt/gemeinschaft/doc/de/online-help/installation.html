Hinweise zur Installations-CD von Gemeinschaft

Am einfachsten können Sie Gemeinschaft installieren indem Sie das ISO-Image der Installations-CD runterladen, auf eine CD brennen und starten. Es handelt sich dabei um eine Netzwerk-Installations-CD, d.h. eine funktionierende Internet-Verbindung wird vorausgesetzt.

DHCP

Viele Anwender haben vermutlich einen DHCP-Server in ihrem Router aktiviert. Das Gemeinschaft-System erhält dadurch automatisch eine IP-Adresse.

Virtuelle Maschine

Selbverständlich kann man die Installations-CD auch auf einer virtuellen Maschine wie VMware o.ä. ausprobieren. Bitte achten Sie dann aber darauf die Netzwerkverbindung auf bridged einzustellen, nicht auf NAT oder host-only.

Manuelle Installation

Sollte Ihr Rechner nicht vom automatischen Installer unterstützt werden (weil er z.B. kein x86 ist) können Sie Gemeinschaft natürlich auch von Hand installieren. Siehe Installation der aktuellsten Entwicklungsversion.
