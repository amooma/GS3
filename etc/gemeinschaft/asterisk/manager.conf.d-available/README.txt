[de]
Um einen Asterisk-Manager-Interface-(AMI)-Account zu aktivieren
benutzen Sie bitte die Web-Oberflaeche oder folgende Shell-Befehle:

cd ../manager.conf.d-enabled
ln -s ../manager.conf.d-available/xy.conf


[en]
To enable an Asterisk Manager Interface (AMI) account please
use the web GUI or the shell commands in following:

cd ../manager.conf.d-enabled
ln -s ../manager.conf.d-available/xy.conf
