[de]
Um einen Asterisk-Manager-Interface-(AMI)-Account zu aktivieren
benutzen Sie bitte die Web-Oberfläche oder folgenden Shell-Befehl:

ln -s ../manager.conf.d-available/xy.conf

bzw. folgenden Shell-Befehl um einen Account zu deaktivieren:

rm xy.conf

Löschen Sie aber keinesfalls die Datei ingore.conf!


[en]
To enable an Asterisk Manager Interface (AMI) account please
use the web GUI or the shell command in following:

ln -s ../manager.conf.d-available/xy.conf

resp. the shell commands in following to deactivate an account:

rm xy.conf

However do not delete ignore.conf!
